Project Title: Sales Data Analysis Dashboard (SQL + Tableau)

Overview: Created artificial sales data set in Snowflake and employed data cleaning using SQL queries (joins, CTEs, aggregations).  Connected data to Tableau to display KPIs and business insights showcasing top customers, top products, and revenue over time. 

Tools Used: Snowflake, SQL, Tableau

Goal: Transform raw sales data into actionable insights by tracking key performance indicators (KPIs), customer spending behavior, and product performance through interactive dashboards.
